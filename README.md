# 📝 TodoList

 A simple and interactive React⚛️-based Todo List app that lets users ➕Add, ✏️Edit, 🗑️Delete, ✅Complete, and ❌Uncomplete tasks. The app also has a Dark🌙and light mode toggle.

 # 🪄Features

- ➕ New tasks

- ✏️ Existing tasks

- 🗑️ Tasks

- ✅ Tasks as completed

- 🔄 Move completed tasks back

- 🌙 Toggle

 # 🛠️ Technologies Used

- ⚛️ React.js

- 🎨 Tailwind CSS

- 📜 JavaScript
